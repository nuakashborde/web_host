Emergency Calling Feature

Electrical Engineering for Wireless Systems
Summer 2020

Emergency Calling (eCall) is a new safety feature in vehicles which automatically calls 112 (Europe’s 911 equivalent) if the car crashes. In addition to setting up a voice call, eCall automatically sends emergency services the car’s airbag data, impact sensor readings, and GPS coordinates in case the driver is unresponsive.

If the vehicle’s crash sensors are not activated, eCall provides an easy “SOS Button” for the driver to directly contact emergency services. According to the European Commission, eCall can speed up emergency services response times by up to 50% in rural areas .

I helped design and validate the electronic system behind eCall with my mentor at Tesla. This system uses the car’s sensor network for crash detection and sets up a phone link with the nearest emergency center using the car’s wireless system. I helped build and test digital and analog circuitry, tested communication protocols between devices, measured power consumption, wrote scripts for stress-testing the system, and analyzed log files. I also got to participate in crash tests for system validation.
