Tesla

Factory Fix Improvement

Internship Project

Assembly Line Fix Improvement

Hardware Debugging
Summer 2020

Background

Tesla vehicles are built on long assembly lines composed of stations. At each station, a worker and/or attaches or fixes a specific part of the car. Each station also has a touchscreen computer for monitoring the production status and inventory for that particular build. If there is any fault at a build station, the computer signals for the entire assembly line to halt. During the summer of 2019, all computer stations on a particular line at Tesla's Fremont factory were upgraded as part of ongoing maintanance.

Problem

After the station upgrades, it was found that certain computers along this assembly line would regularly crash, causing faults which stopped the production line. To prevent this fault, Tesla reverted back to an older, touchscreen station which was much more expensive to maintain.

Solution

By taking apart the faulty touchscreen stations, I was able to narrow down the defective component through process of elimination. I found that the issue was on the circuit board which housed the computer. On the PCB itself, I systematically isolated working parts of the circuit by depopulating components, used thermal imaging to identify overheating, and monitored signal communication to identify that two components were used incorectly in the board's design.

Once I identified these problems, I created a reworked PCB with correct changes and tested it in a standalone computer. With all changes validated, I created a new PCB with the incorporated fixes and deployed the solution back into the factory.

My solution was able to reuse existing Tesla scrap materials and save the company $500,000.
